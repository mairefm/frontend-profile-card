## Sobre o projeto
Projeto simples e moderno em **HTML e CSS** utilizado para ensinar conceitos básicos de estilização e responsividade nas minhas aulas de Frontend no **Colégio Dom Bosco**.

Este card apresenta:
- Foto de perfil  
- Nome e função  
- Ícones sociais  
- Layout centralizado  
- Estilização com sombras, bordas suaves e cores modernas  
- Estrutura de projeto organizada  

É ideal para alunos que estão dando os primeiros passos no Frontend.

---

## Tecnologias usadas
- **HTML5**
- **CSS3**
- Google Fonts  
- Deploy com **GitHub Pages**
